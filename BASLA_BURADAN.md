## 📱 COLAB'DA NASIL ÇALIŞTIRILACAK?

1. **Google Colab'ı aç:**
   ```
   https://colab.research.google.com
   ```

2. **GitHub'daki OLKA_Dashboard.ipynb dosyasını aç:**
   - File → Open notebook
   - GitHub sekmesini seç
   - URL'sini yapıştır: `https://github.com/YOUR_USERNAME/olka-sprint-dashboard`
   - OLKA_Dashboard.ipynb'yi seç

3. **Hücreleri çalıştır:**
   - İlk hücreye tıkla
   - Shift+Enter tuşu basıp çalıştır
   - Tüm hücreleri sırasıyla yap

4. **ngrok URL'sini kopyala:**
   - Colab çıktıda bir link verecek
   - Tarayıcıda aç
   - **Dashboard açılır!** 🎉
